이미지 가이드

기본 규칙

  • 색상 정보를 가진 영역만 정확히 잘라내는 것을 기본으로 하되, 개발이나 코딩상의 이슈로 규격을 가지는 경우 여백을 두고 슬라이싱하는 것을 허용한다.
  • 포토샵 Save for Web 메뉴에서 이미지에 최적화된 포맷으로 저장하여 성능 향상을 도모한다.

이미지 압축 포맷

GIF

  • 256색 이하의 색상으로 표현되는 아이콘, 이미지.
  • 애니메이션 이미지 또는 투명한 부분까지 저장이 가능하나 반투명한 알파값은 표현 불가.
  • 투명한 부분과 이미지의 경계가 뚜렷한 경우 사용 가능

JPG

  • 사진, 그라데이션 등 많은 색상 정보를 포함하는 이미지에 적합.
  • 육안으로 화질의 손실이 느껴지지 않는 선에서 최대한 압축률을 높여 저장하는 것을 권장 (60~80)

PNG

  • 알파값 표현이 가능하므로 투명/반투명한 이미지를 저장할 때 적합
  • 화질의 손실이 거의 없는 반면, 압축률이 낮은 편이므로 꼭 필요할 때 사용

이미지 네이밍

이미지 파일명 지정은 네이밍 규약 공통 규칙 및 이미지 네이밍 규칙에 의거한다.

Sprites 이미지 규격

Sprites 기법에 사용하는 이미지는 다음과 같은 규격을 따른다.
각 개체간 간격은 최소 1px 이상 떨어져있어야 한다.

타이틀 이미지

  • 수직정렬형 이미지 스프라이트를 활용한다.
  • 각 개체의 기본간격을 50px로 하되, 높이가 작을 경우 25px단위로 줄인다.

버튼 이미지

  • 바둑판형 이미지 스프라이트를 활용한다.
  • 위치지정을 10px 단위로 지정할 수 있도록 이미지를 배치시킨다.

아이콘 이미지

  • 가능하면 바둑판형 스프라이트를 기본으로 활용한다.
  • 우측 배치되는 아이콘이 있는 경우, 수직정렬형 스프라이트를 활용하며 좌측에 배치되는 아이콘은 가장 좌측에, 우측으로 배치되는 아이콘은 가장 우측으로 배치한다.
  • 배경 위치지정을 25px 단위로 지정할 수 있도록 이미지를 배치시킨다.

롤오버 이미지

  • off 이미지 하단에 on 이미지를 배치시킨다.
