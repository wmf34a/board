기본 규칙

권장 환경

DOCTYPE

HTML 문법은 해당 DTD의 명세에 기반하되, 확장성 및 호환성을 고려하여 XHTML1.0 기준으로 작성한다.

일반 웹일 경우 XHTML1.0를 적용한다

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

모바일 웹일 경우 HTML5를 적용한다

<!DOCTYPE html>

DOCTYPE은 문서의 최상단에 위치해야 하며, 그 위에 주석이나 스크립트 등의 코드가 오게해서는 안된다.

언어 코드 : ko

다국어 작업시 국가별 언어 코드 참조

<html xmlns="http://www.w3.org/1999/xhtml" lang="ko" xml:lang="ko">

인코딩 : utf-8

다국어 작업시 국가별 인코딩 참조

<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />

HTML 기본 문서

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml" lang="ko" xml:lang="ko">
<head>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
<meta http-equiv="X-UA-Compatible" content="IE=edge" />
<title>화면명 &lt; 대메뉴명 | 가이스트코리아</title>
<link rel="stylesheet" type="text/css" href="../css/default.css" media="all" />
<link rel="stylesheet" type="text/css" href="../css/common.css" media="all" />
<script type="text/javascript" src="../js/jquery-1.9.1.min.js" charset="utf-8"></script>
<script type="text/javascript" src="../js/ui.js" charset="utf-8"></script>
</head>
<body>
</body>
</html>

자주 틀리는 문법

모든 엘리먼트와 애트리뷰트는 소문자로 작성한다.

<P Class="desc">유의사항</P>				(X)
<p class="desc">유의사항</p>				(O)
<a href="#" onClick="" onMouseOver="">유의사항</a>	(X)
<a href="#" onclick="" onmouseover="">유의사항</a>	(O)

태그는 반드시 닫는다.

<input type="text" id="uid">				(X)
<input type="text" id="uid" />				(O)

범위가 없는 다음 태그는 특히 유의한다.

area, br, col, frame, hr, img, input, link, meta, param

태그는 중첩 순서에 따라 닫는다.

<p><strong>유의사항</p></strong>				(X)
<p><strong>유의사항</strong></p>				(O)

애트리뷰트는 큰 따옴표( " )로 묶는다.

<img src='images/sample.gif' alt='예시' />		(X)
<img src="images/sample.gif" alt="예시" />		(O)

다음 특수문자는 Entity 코드로 표기한다.

& → &amp;	< → &lt;	> → &gt;	" → &quot;

URL을 입력하는 과정에서 &를 코드로 치환하지 않는 실수가 자주 발생한다.

필수 애트리뷰트

필수 애트리뷰트는 누락시 문법 오류로 검출되므로, 값이 없을 경우라도 누락해서는 안된다.

흔히 누락하는 애트리뷰트

<img src="" alt="" />
<table border="1" summary=""></table>
<form action=""></form>
<iframe src="" title=""></iframe>
<frame title="" />

개발상의 이유로 비어있는 프레임을 사용할 경우 title="빈 프레임"으로 기입한다.

<ifrmae src="sample.html" title="빈 프레임" scrolling="no"></iframe>

오류 점검

코딩 후 W3C Validation 또는 firefox 부가기능 HTML Validator에서 오류건수 0이 되도록 검증한다.
