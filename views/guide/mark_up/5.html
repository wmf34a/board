CSS 가이드

기본 규칙

설정

외부 스타일시트(<link />)를 사용한다. (@import는 일부 브라우저에서 HTML 문서 로딩 후에야 로드됨)

<link rel="stylesheet" type="text/css" href="../css/default.css" media="all" />

단, 인풋요소 및 테이블 셀 등 페이지마다 변하는 객체에 개별 사이즈를 지정할 때는 인라인 스타일을 허용한다.

<input type="text" style="width:200px">

CSS 파일 최상단에 언어셋을 지정한다.

@charset "utf-8"

정렬

  • 한가지 스타일은 한 줄에 작성하며, 줄바꿈 시 빈 줄이 생기지 않게 한다.
  • CSS 유닛간 빈 줄을 한 줄까지 허용한다.
  • 띄어쓰기는 스페이스 1칸만을 허용한다. 임의로 2칸 이상 띄어쓰지 않는다.
  • 각 속성간은 한 칸 띄우며, 속성과 값 사이는 띄어쓰지 않는다.
    예) 선택자 {속성:값; 속성:값;}
  • 자식 선택자(>) 인접 선택자(+) 사용시 붙여쓴다.
    예) #snb>li.on>a
  • 들여쓰기는 하지 않음을 원칙으로 하며, 일부 웹폰트나 벤더프리픽스를 붙여 스타일링하는 경우에는 경우에만 가독성을 위한 줄바꿈 및 들여쓰기를 허용한다.

스타일 선언

  • 크로스브라우징을 위한 핵은 최대한 지양한다.
  • 속성 선언에 따옴표는 사용하지 않는다. 단, 한글폰트 선언이나 공백표현처럼 사용이 필요한 경우는 예외되며 사용 시에는 홑따옴표(')로 통일한다.

CSS 속성 선언 순서

표시 - 배치 - 사이즈 - 폰트 - 기타 - 배경 - 상태 순서로 기술한다.

CSS 속성 선언 순서
순서 속성 의미 비고
1 display 표시  
2 float 흐름  
3 position 위치  
4 width & height 크기  
5 margin & padding 간격  
6 border 외곽선  
7 font 글꼴 축약형은 쓰지않으며, color, font-weight, font-size, line-height, font-family 순으로 기입
8 etc 기타 text-align, vertical-align, text-indent, white-space, text-decoration 등
9 background 배경 축약형으로 표기하되, 리셋 등 필요한 경우 분리하여 작성
10 overflow 넘침  
11 z-index 정렬 유지보수시 편의를 위해 z-index:10과 같이 차등값을 10 단위로 설정하며, 콘텐츠 내의 z-index값은 최대 1000을 넘지 않도록 한다.

주석

  • 주석으로 CSS 유닛 상단에 모듈명 또는 화면명을 명료하게 기입한다.
  • CSS 주석 기호( /*, */ )와 내용 사이에는 반드시 공백 한 칸이 있어야 한다.
/*로그인*/				(X)
/* 로그인 */				(O)

default.css

  • 강조표시된 부분은 환경에 따라 수정하여 사용한다.
  • 간격 설정은 개발상의 이슈시 편의를 위해 적절히 설정해두되, 가급적 퍼블리싱 구축시에는 사용하지 않는다. 수치를 기입하는 방식은 인라인 스타일과 다를바 없으며, 추후 변경시 CSS만으로 컨트롤이 불가해지기 때문이다.
