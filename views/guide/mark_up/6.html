스타일링 기법

IR 기법

정보를 포함하는 이미지(인포그래픽)를 배경처리한 뒤, 전경에 마크업한 대체콘텐츠는 숨기는 기법이다. alt값 만으로는 대체콘텐츠를 온전히 표현할 수 없는 경우 longdesc 대용으로 사용한다. longdesc는 별도의 대체콘텐츠 파일을 링크하는 방식으로 처리하지만 IR기법에서는 동일 페이지 상에 처리함으로써 개발 편의와 사용성 증대를 도모할 수 있다는 이점이 있다.

여러가지 방식의 IR 기법이 있는데, 가이스트코리아 프로젝트에서는 다음과 같은 방식으로 처리한다.

단순 이미지의 경우

HTML

<p><img src="img_festival.jpg" alt="선사문화축제 일정안내 (하단설명)" /></p>
<div class="hidden">
	<h1>선사문화축제</h1>
	<p>2015.10.09(금) ~ 10.11(일)</p>
</div>

CSS

/* default.css */
.hidden {position:absolute; left:-9999px; width:1px; height:1px; font-size:0; line-height:0; overflow:hidden;}

이미지에 버튼 등의 조작요소가 포함된 경우

HTML

<div class="ir_wrap">
	<p><img src="img_festival.jpg" alt="선사문화축제 일정안내 (하단설명)" /></p>
	<h1 class="hidden">선사문화축제</h1>
	<a href="#" target="_blank" title="새 창" class="ir_btn btn_shortcut">홈페이지 바로가기</a>
	<p class="hidden">2015.10.09(금) ~ 10.11(일)</p>
</div>

CSS

/* default.css */
.hidden {position:absolute; left:-9999px; width:1px; height:1px; font-size:0; line-height:0; overflow:hidden;}

/* common.css */
.ir_wrap {position:relative;}
.ir_wrap .ir_btn {display:block; position:absolute; text-indent:100%; white-space:nowrap; background:url(../images/common/blank.gif) no-repeat; overflow:hidden;}

/* content.css */
.btn_shortcut {top:150px; left:50px; width:100px; height:30px;}

Sprites 기법

이미지 리퀘스트 회수를 줄여 성능향상을 도모한다. 이미지 규격은 이미지 슬라이싱 기준에 맞추도록 한다.

HTML

<ul class="sns_share">
	<li><a href="#" class="sp_ico tw">트위터</a></li>
	<li><a href="#" class="sp_ico fb">페이스북</a></li>
	<li><a href="#" class="sp_ico yt">유튜브</a></li>
</ul>

CSS

/* common.css */
.sp_ico {display:inline-block; text-indent:100%; white-space:nowrap; background:url(../images/common/sp_ico.gif) no-repeat; overflow:hidden;}

/* content.css */
.sns_share {overflow:hidden;}
.sns_share li {float:left; margin:0 5px;}
.sns_share .sp_ico {width:30px; height:30px;}
.sns_share .sp_ico.tw {background-position:0 0}
.sns_share .sp_ico.fb {background-position:-40px 0}
.sns_share .sp_ico.yt {background-position:-80px 0}
